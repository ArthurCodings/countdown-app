# 音频文件说明

这个目录需要包含以下音频文件来提供完整的音效体验：

## 必需的音频文件

### 1. 滴答音效 (tick.mp3 / tick.ogg)
- **用途**: 时钟秒针跳动音效
- **建议**: 轻柔的滴答声，时长约0.1-0.2秒
- **推荐音量**: 中等，不应过于突出
- **免费资源**: [Freesound.org](https://freesound.org) 搜索 "tick" 或 "clock"

### 2. 心跳音效 (heartbeat.mp3 / heartbeat.ogg)
- **用途**: 倒计时最后一分钟的紧张氛围音效
- **建议**: 模拟心跳的低频音效，时长约0.8-1秒
- **推荐音量**: 中低音量，营造紧张感
- **免费资源**: [Freesound.org](https://freesound.org) 搜索 "heartbeat"

### 3. 完成提示音 (completion.mp3 / completion.ogg)
- **用途**: 倒计时完成时的庆祝音效
- **建议**: 愉悦的铃声或成功提示音，时长1-3秒
- **推荐音量**: 较高，用于吸引注意
- **免费资源**: [Freesound.org](https://freesound.org) 搜索 "success" 或 "achievement"

## 音频格式要求

- **主要格式**: MP3 (兼容性最好)
- **备用格式**: OGG (更好的压缩率)
- **采样率**: 44.1kHz 或 48kHz
- **比特率**: 128kbps 或更高
- **文件大小**: 建议每个文件小于500KB

## 获取免费音效的推荐网站

### 1. Freesound.org
- **特点**: 最大的免费音效库
- **许可**: Creative Commons许可
- **使用**: 需要注册账号，搜索并下载

### 2. Zapsplat.com
- **特点**: 高质量音效库
- **许可**: 免费个人使用
- **使用**: 需要注册账号

### 3. Adobe Audition 音效库
- **特点**: 专业质量
- **许可**: 如果有Adobe订阅可免费使用

### 4. 自己录制
- **滴答声**: 使用节拍器或真实时钟录制
- **心跳声**: 在安静环境下录制真实心跳
- **提示音**: 使用音乐软件生成简单铃声

## 音频处理建议

如果需要编辑音频文件，推荐使用：

1. **Audacity** (免费)
   - 跨平台音频编辑器
   - 可以调整音量、长度、淡入淡出等

2. **在线工具**
   - Online Audio Converter
   - AudioMass (在线Audacity)

## 音效设置指南

1. **音量平衡**:
   - 滴答声: 50-60% 
   - 心跳声: 40-50%
   - 完成音: 70-80%

2. **淡入淡出**:
   - 所有音效都应该有轻微的淡入淡出处理
   - 避免突然开始或结束的声音

3. **循环处理**:
   - 滴答声和心跳声需要能够无缝循环
   - 确保音频开头和结尾平滑衔接

## 版权注意事项

- 确保使用的音效具有适当的使用许可
- 如果用于商业用途，请确认许可证允许商业使用
- 建议在项目中注明音效来源和许可信息

## 备选方案

如果无法获取音频文件，应用会在没有音效的情况下正常运行：

- 音效按钮会显示但不播放声音
- 所有视觉反馈仍然可用
- 用户可以随时添加音频文件启用音效

## 文件放置

将下载的音频文件放置在以下位置：
```
assets/
  audio/
    tick.mp3
    tick.ogg
    heartbeat.mp3
    heartbeat.ogg
    completion.mp3
    completion.ogg
```

## 测试音效

在浏览器中打开应用后：
1. 点击音效控制按钮启用音效
2. 调整音量到合适水平
3. 在设置中测试各种音效开关
4. 确保倒计时最后阶段心跳音效正常
5. 测试倒计时完成时的提示音

如有任何问题，请检查浏览器控制台的错误信息。
